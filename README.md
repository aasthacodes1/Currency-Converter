# Currency Converter

A simple and user-friendly Currency Converter web application built using HTML, CSS, and JavaScript. It allows users to convert one currency into another using real-time exchange rates from an API.

## Technologies Used
- HTML
- CSS
- JavaScript
- Exchange Rate API

## Features
- Convert between multiple international currencies
- Real-time exchange rates
- Easy-to-use interface
- Swap currencies option
- Instant conversion results

## How It Works
1. Select the source currency.
2. Select the target currency.
3. Enter the amount to convert.
4. Click the Convert button.
5. View the converted amount instantly.
